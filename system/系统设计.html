# 数据库字段

字段名 类型 备注 取值
organ_uuid chars 主键id 非null
organ_code chars 代码
organ_type chars 类型
in_use char1 是否可用
parent_uuid chars 父组织主键
status char1 是否
modifierid chars 修改人Id
modtime datetime 修改时间
description chasrs 描述
account_code chars 资金账户code
shortname chars 简称
is_parent tinyint 是否有子节点
